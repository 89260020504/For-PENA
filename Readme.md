### Задание находится в файле task.txt

main.php Запускает скрипт расчета цены для двух операторов доставки.

Как показывает собственный опыт, лучше использовать фабрику. Расчет цены лучше реализовать для оператора доставки в его собственном классе (редко бывает, что все ложится под один шаблон)

>На этом этапе я бы оставил структуру такой (KISS), при появлении доп данных (например: наличие внешнего апи у операторов или дополнительных операторв), можно сделать абстрактный класс из интерфейса CarrierInterface и добавить туда универсальую функцию для расчета тарифа при необходимости изменяя логику в соответствующих классах (папка Carriers).
